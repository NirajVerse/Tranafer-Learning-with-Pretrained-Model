# Dry vs Moist Classification using Xception

A deep learning project using the Xception model to classify images as dry or moist.

## Contents

- `xception_dryMoist.ipynb` - Main Jupyter notebook with model implementation and analysis

## Model

This project uses the Xception architecture for binary classification.

## Requirements

- TensorFlow/Keras
- Python 3.x
- Jupyter Notebook
